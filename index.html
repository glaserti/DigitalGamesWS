Ihr Digital Game in Unterricht und Lehre

Sind Sie selbst passionierte_r Gamer_in und haben sich schon oft vorgestellt, wie es wäre, digitale Spiele in Ihren Unterricht einzubinden? Diese Veranstaltung bietet Ihnen die Möglichkeit dazu!

Workshop-Details

  • Ort: Creative Space im Club Behring, Bahnhofstraße 7, Marburg
  • Datum: Donnerstag, 28.11.2024
  • Zeit:
    • Interaktive Geschichten: 9:00 bis 12:00
    • Point-and-Click Adventures: 13:00 bis 17:00

Workshop 1: Interaktive Geschichten mit Inky

Dieser Workshop zeigt Ihnen, wie Sie eine interaktive Geschichte erstellen können, in der die Leser die Handlung durch ihre Entscheidungen beeinflussen.

  • Schreiben Sie eine Kurzgeschichte in Inklwriter
  • Erstellen Sie eine komplexe Geschichte mit Ink-Skriptsprache
  • Exportieren Sie das Spiel als HTML-Datei
  • Passen Sie das Erscheinungsbild mit HTML und CSS an

Voraussetzungen: Bitte installieren Sie Inky auf Ihrem Rechner: Download Inky

Workshop 2: Point-and-Click Adventures mit CreATe

Erstellen Sie ein eigenes Point-and-Click Adventure ohne Programmierkenntnisse mit dem Tool CreATe.

  • Erstellen Sie eine Spielewelt mit interaktiven Objekten
  • Fügen Sie Aufgaben wie Multiple-Choice und Drag-and-Drop hinzu

Voraussetzungen: Installieren Sie Active Presenter: Download Active Presenter
